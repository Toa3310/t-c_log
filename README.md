# tascon（仮称）

学習者向けの「タスク管理 × その日の調子の記録」アプリです。  
その日の状態に合わせて学習を進め、達成状況や学習傾向を記録・可視化することを目的としています。

現在はFlaskを用いたWeb版を中心に開発しており、CLI版は初期プロトタイプとして残しています。

※ アプリ名は検討中です。現時点の第一候補として「tascon」を使っています。

---

## 概要

資格勉強・大学学習・自主学習などを行う人向けに、

- 今日やること
- どれくらい進んだか
- どんな状態だったか

をシンプルに記録できるアプリです。

一般的なTodoアプリとは異なり、  
「コンディション（集中度・気分）」と「学習行動」を結びつけることを重視しています。

---

## 開発目的

資格勉強や大学の自主学習を進める中で、

- 既存のタスク管理アプリは入力項目が多く、続けるのが面倒
- そのときの集中度や気分によって学習の進め方を変えたいのに、対応できるアプリがない

と感じることがありました。

そこで、自分が無理なく続けられる学習ログを作ろうと思い、
軽量で扱いやすいアプリとして開発を開始しました。

---

## 主な機能

### 実装済み
- コンディション選択（よくない / 普通 / 良い）
- コンディションに応じたテンプレートタスクの提案
- タスクの選択・完了チェック
- 進捗率表示（0〜100%）
- 途中終了（そこまでの記録を残せる）
- 一言メモ（その日の振り返り）
- SQLite への記録の永続化
- これまでの記録の一覧表示・削除

### 予定
- 気分記録
- 達成率・コンディションのグラフ可視化
- コンディションと達成率の関連確認

---

将来的に使ってもらえるアプリを目指しています。
