# TaskCondition

学習者向けの「タスク管理 × コンディション記録」アプリです。  
その日の状態に合わせて学習を進め、達成状況や学習傾向を記録・可視化することを目的としています。

現在はCLI版をベースに、Flaskを用いたWebアプリ化を進めています。

---

## 概要

資格勉強・大学学習・自主学習などを行う人向けに、

- 今日やること
- どれくらい進んだか
- どんな状態だったか

をシンプルに記録できるアプリです。

一般的なTodoアプリとは異なり、  
「コンディション（集中度・気分）」と「学習行動」を結びつけることを重視しています。

---

## 開発目的

資格勉強や大学の自主学習を進める中で、

- 既存のタスク管理アプリは入力項目が多く、続けるのが面倒
- そのときの集中度や気分によって学習の進め方を変えたいのに、対応できるアプリがない

と感じることがありました。

そこで、自分が無理なく続けられる学習ログを作ろうと思い、
軽量で扱いやすいアプリとして開発を開始しました。

---

## 主な機能（予定含む）

### タスク管理
- コンディションに応じたテンプレートタスクの提案（CLI版で実装済み）
- タスクの選択・完了チェック
- 進捗率表示（0〜100%）

### Webアプリ化（開発中）
- コンディション選択画面
- タスク選択・表示
- セッションによる一時的なデータ保持

### コンディション記録
- コンディション記録（いい,普通,よくない）
- 気分記録
- 一言メモ

### 学習ログ（予定）
- 日ごとの学習記録保存（DB導入予定）
- 達成率確認
- コンディションとの関連確認

---

## 想定ユーザー

まずは自分自身が使うために開発しています。  
同じように「学習の継続」や「コンディションに応じた進め方」に悩む人にも、  
将来的に使ってもらえるアプリを目指しています。
